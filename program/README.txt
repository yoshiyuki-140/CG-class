テキストのプログラムとデータについての説明書

【留意事項】
・Linuxの場合にディレクトリと称するものはWindowsではフォルダと称する．
・ディレクトリ（フォルダ）およびファイル名の英文字には大文字小文字を使用しているので，もし，小文字しか存在しない場合は，当該圧縮ファイルの展開時にファイル名を自動的に小文字にしている可能性がある．
・本説明書およびMakefileなどはWindows上で記述したため，改行コードがLinuxと異なる場合があり，これは，Linuxでコンパイルエラーの原因の一つとなるため，各自で改行コードの訂正を願います．

【ディレクトリ（フォルダ）名の説明】
ディレクトリ（フォルダ）は次が存在する．
C               2章から7章のC言語プログラム，PPMデータ，Makefileが存在する．
C++Billiard     8章2節　ビリヤードのC++言語プログラム，PPMデータ，Makefileが存在する．
C++Pendulum     8章3節　振り子のC++言語プログラム，Makefileが存在する．
C++Boids        8章4節　魚の群れのC++言語プログラム，Makefileが存在する．
C++Hand         8章5節　手の動きのC++言語プログラム，Makefileが存在する．
C-Dog           8章6節　犬の歩行のC言語プログラム，Makefileが存在する．

【ファイル名の説明】
・プログラム名に英語の大文字小文字を用いている．
・プログラム名の接頭にある"pn-"のnは章番号を表している．
